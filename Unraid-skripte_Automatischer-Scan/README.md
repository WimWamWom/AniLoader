# Unraid Skripte - Automatischer Scan

Diese Skripte sind **optional** und nur für Unraid-Nutzer relevant, die den automatischen Medienscan nutzen möchten.

## Skripte

| Skript | Beschreibung |
|--------|--------------|
| `check-new.sh` | Prüft auf neue Downloads und triggert Medienscan |
| `check-german.sh` | Prüft auf deutsche Sprachversion |
| `last_run_summary.sh` | Zeigt eine Zusammenfassung des letzten Laufs |
